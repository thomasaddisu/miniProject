# MyShop - Simple E-commerce App

This is a small React-based e-commerce application that allows users to browse products, add them to a cart, adjust quantities, and remove items. It demonstrates React state management, routing, and basic UI with Tailwind CSS.

## Features

- Display products with image, name, category, price, and description.
- Add products to cart.
- Update product quantity (+ / -) in the cart.
- Remove products from the cart.
- Dynamic total price calculation.
- Header shows live cart item count.
- “Added to cart” message appears briefly after adding an item.
- Empty cart displays a friendly message.

## Tech Stack

- React
- React Router
- Tailwind CSS
- JavaScript
