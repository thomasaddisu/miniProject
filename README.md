# MyShop - Simple E-commerce App with ChatBot

This is a React-based e-commerce application that allows users to browse products, add them to a cart, manage quantities, and remove items. It also includes an **interactive rule-based chatbot** to assist users with gift ideas, tech gadgets, and cart information.

## Features

- Browse products with **image, name, category, price, description, and stock info**.  
- **Add products to cart** and see live quantity updates.  
- **Increase / decrease product quantity** directly in the cart.  
- **Remove products** from the cart.  
- **Dynamic subtotal, tax, and total price calculation**.  
- Header shows **live cart item count**.  
- "Added to cart" and "Out of stock" messages appear briefly for feedback.  
- **Empty cart displays a friendly message**.  
- **Floating chatbot** at the bottom-right corner:  
  - Suggests **birthday gifts, anniversary gifts, tech gadgets, and other products**.  
  - Displays **product images, prices, and "Add to Cart" buttons** directly in chat.  
  - Shows **cart contents with images and totals**.  
  - Auto-scrolls to newest messages for smooth UX.  

## Tech Stack

- **React** for UI and state management  
- **React Router** for routing between pages  
- **Tailwind CSS** for styling  
- **JavaScript** for interactivity and chatbot rules
